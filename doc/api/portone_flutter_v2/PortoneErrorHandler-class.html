PortoneErrorHandler

PortoneErrorHandler class

A callable class that wraps user-provided error callbacks, allowing them to accept either a single Object? error parameter or both Object? error and StackTrace? stackTrace. Internally, it normalizes the callback to void Function(Object? error, [StackTrace? stackTrace]).

The provided handler must be either:

  • void Function(Object? error), or
  • void Function(Object? error, StackTrace? stackTrace).

Example:

// One-arg callback:
final handlerA = PortoneErrorHandler((error) {
  print('Error: $error');
});

// Two-arg callback:
final handlerB = PortoneErrorHandler((error, stack) {
  print('Error: $error\nStackTrace: $stack');
});

// Always called with both parameters (second may be null):
handlerA('Oops', null);
handlerB('Oops', StackTrace.current);
