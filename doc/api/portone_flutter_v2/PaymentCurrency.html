PaymentCurrency

PaymentCurrency enum

ISO 4217 기반의 결제 통화 enum입니다.

ISO 4217은 각 통화에 대해 고유한 3자리 알파벳 코드(예: KRW, USD, EUR 등)를 부여하며, 은행 시스템, 결제 게이트웨이, 금융 API 등에서 사용됩니다.

직렬화 시에는 value 필드를 사용하여 문자열 값을 제공합니다.

참고: ISO 4217 위키백과
